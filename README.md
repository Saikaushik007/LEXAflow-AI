# LEXAflow AI

## Problem
Agents handling complex cases must manually search policies, causing delays and compliance risks.

## Solution
LEXAflow AI provides context-aware policy suggestions inside Appian workflows using AI.

## Features
- Context-aware retrieval
- Source citations
- Prototype implementation

## Tech Stack
- Python
- NLP Models
- Vector Search

## Note
This is a prototype created for Appian AI Hackathon.
